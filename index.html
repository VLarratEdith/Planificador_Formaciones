Configurar Firebase

Para sincronizar tus datos entre dispositivos necesitas un proyecto gratuito de Firebase. Sigue estos pasos:

1

Crear proyecto

Ve a console.firebase.google.com y crea un proyecto nuevo (puedes dejar el nombre por defecto).

2

Activar Authentication

En el menú lateral entra en Build > Authentication. Haz clic en "Get started" y activa el proveedor Email/Password. Guarda los cambios.

3

Crear base de datos Firestore

En Build > Firestore Database, haz clic en "Create database". Elige Start in test mode y selecciona una región cercana (ej: europe-west1). Espera a que se cree.

4

Obtener la configuración

En Project settings (icono de engranaje arriba a la derecha), desplázate hasta "Your apps" y haz clic en el icono </> (Web). Registra la app y copia el objeto firebaseConfig.

5

Pegar la configuración

Abre el código fuente de esta página (Ctrl+U) y busca FIREBASE_CONFIG al inicio del script. Reemplaza los valores con los que copiaste. El resultado debe ser algo así:

const FIREBASE_CONFIG = { apiKey: "AIzaSy...", authDomain: "mi-proyecto.firebaseapp.com", projectId: "mi-proyecto-id", storageBucket: "mi-proyecto.appspot.com", messagingSenderId: "123456789", appId: "1:123456789:web:abc..." };
6

Recargar la página

Guarda el archivo y recarga la página. Ya podrás registrarte e iniciar sesión.

Mientras tanto, la app funciona en modo local (los datos solo se guardan en este navegador).
